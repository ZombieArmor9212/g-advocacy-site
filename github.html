Proyekto at Arkitektura

Ang website na ito ay binuo gamit ang custom-coded assets. Ang repository na ito ang nagsisilbing "utak" ng site, kung saan nakaimbak ang lahat ng styling (CSS), interactive logic (JS), at structure (HTML) na hindi kayang suportahan ng standard Google Sites editor.

Bakit GitHub?

Gumagamit tayo ng GitHub upang malampasan ang limitasyon ng Sites app. Dahil ang ating disenyo ay nangangailangan ng Glassmorphism at Independent In-Out Transitions, kailangan nating i-host ang code sa labas at i-embed ito upang mapanatili ang premium na itsura ng adbokasiya.
