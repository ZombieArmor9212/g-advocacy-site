Teknikal na Hamon

Mahirap ang manu-manong pag-code dahil kailangan ng matinding lohika at tiyaga. Binuo namin ito mula sa scratch para masiguro na pulido ang bawat detalye ng aming website.

Kalidad ng Resulta

Dahil kami ang may kontrol sa code, mas mabilis at mas "smooth" ang website na ito. Hindi ito limitado ng mga template, kaya mas maganda ang karanasan ng mga bibisita.

Proyekto at Arkitektura

Ang repository na ito ang nagsisilbing "brain" ng proyekto. Dito nakasulat ang lahat ng utos (CSS at JS) na nagpapatakbo sa disenyo at mga interactive na bahagi ng site.

Bakit GitHub?

Dito namin iniimbak ang aming code para maging "open source." Pinili namin ang GitHub para magkaroon ng paraan na ma-apply ang mga advanced features na wala sa basic tools.
