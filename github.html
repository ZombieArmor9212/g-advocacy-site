Pinapagana ng GitHub Repository

Ano ang nilalaman nito?

Ang repository na ito ay naglalaman ng mga custom HTML, CSS, at JavaScript na nagbibigay-buhay sa ating website. Dito nanggagaling ang glassmorphism design, ang banig na background, at ang mga smooth na in-out animations ng bawat pahina.

Bakit hindi ma-edit direkta sa Google Sites?

Limitado ang built-in editor ng Google Sites para sa mga basic na disenyo. Tinatanggal o bina-block nito ang mga advanced na animations, custom layouts, at hover effects. Upang mapanatili ang mataas na kalidad at "premium feel" ng ating website, ginagamit natin ang Embed Code feature at kinukuha ang buong disenyo mula sa ating GitHub code.
